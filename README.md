# Explore US Bikeshare Data

## Date Created
January 29, 2026

## Description
This project analyzes bikeshare data for Chicago, New York City, and Washington.
It allows the user to filter data by city, month, and day and then displays:
- Popular travel times
- Common stations and trips
- Trip duration statistics
- User demographics (where available)

## Usage
Run the script using Python from your terminal:
python bikeshare.py

## Files Used
- `bikeshare.py` – main analysis script
- `chicago.csv` – bikeshare data for Chicago
- `new_york_city.csv` – bikeshare data for New York City
- `washington.csv` – bikeshare data for Washington
